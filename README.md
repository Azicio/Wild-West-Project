# 🌿 Bio-Stack: The Wild West
**Status:** Public Sandbox | **Role:** Ecological Taxonomy & Field Logging

A niche software application designed to track species taxonomy, habitat data, and technical Mandarin translations for field researchers.

### 🧬 Features
- **Taxonomic Engine:** Displays full biological hierarchy from Kingdom to Species.
- **Field Notes:** Submit real-time observations directly to a Google Sheets backend.
- **Status Badges:** IUCN-style visual indicators for species conservation status.

### 🧪 Skill Stacking Integration
Combines **Data Analysis** (Taxonomy structures) with **Technical Mandarin** and **Ecology** to maximize field worth.
